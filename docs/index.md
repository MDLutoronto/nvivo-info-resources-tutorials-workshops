# NVivo: Information, Resources, Tutorials, and Workshops

[NVivo](https://lumivero.com/products/nvivo/) is a program used for qualitative data analysis. It supports researchers in managing, organizing, and analyzing qualitative data to produce new insight, infer relationships, and identify themes.
